# Change Log

All notable changes to the Frank!Framework extension will be documented in this file.

## [0.1.1] - 2026-05-11

### Added
- Configuration flow visualization in the secondary sidebar (updates on every save and document change)
- Real-time XML validation for Frank configurations with inline diagnostics
- XPath and JSONPath expression validation
- Frank!Framework Wiki snippets loaded as global autocomplete suggestions, organized by component name
- User-defined snippets with grouping, editing, and a management UI
- Frank!Snippets tree view in the Explorer for browsing and inserting snippets
- Frank!Start tree view for running Frank projects via Ant or Docker Compose
- Run history with one-click re-run of previous projects
- Frank!Framework version management per project (latest, latest stable, or locally installed)
- Create new Frank project from a Simple Frank template
- Frank!Doc hyperlinks for component names in XML configuration files
- Go-to-definition for `sessionKey` attributes
- Find all references for pipe names across configuration files
- Rename support for pipe names (scoped to the enclosing adapter) and session keys (workspace-wide)
